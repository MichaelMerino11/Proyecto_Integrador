Materiales

Para medir la temperatura en el data center de nuestra universidad, utilizamos los siguientes materiales:

  • ESP32: Un microcontrolador que permite conectar sensores y enviar datos a través de Wi-Fi.
  • DHT22: Un sensor de temperatura y humedad que proporciona lecturas precisas.
  • Cables: Utilizados para conectar el ESP32 con el DHT22 y otros componentes.
  • Protoboard: Una placa de pruebas para montar y conectar los componentes electrónicos.
  • Plataforma Firebase: Utilizada para almacenar y visualizar los datos de temperatura en tiempo real.

Pasos para Configurar el Sistema de Medición de Temperatura

  1. Preparar el Material: Reúne todos los materiales necesarios: ESP32, DHT22, cables, protoboard y una cuenta en Firebase.
  2. Conectar el DHT22 al ESP32:
    • Conecta el pin VCC del DHT22 al pin de 3.3V del ESP32.
    • Conecta el pin GND del DHT22 al pin GND del ESP32.
    • Conecta el pin DATA del DHT22 a uno de los pines GPIO disponibles en el ESP32 (por ejemplo, GPIO 4).
  3. Montar el Circuito en la Protoboard:
    • Coloca el ESP32 y el DHT22 en la protoboard.
    • Utiliza cables para conectar los componentes de acuerdo con los pasos anteriores.
  4. Configurar el ESP32:
    • Instala el software de desarrollo adecuado (por ejemplo, Arduino IDE).
    • Configura el entorno de desarrollo para el ESP32.
    • Escribe o carga el código necesario para leer datos del DHT22 y enviar estos datos a Firebase.
  5. Configurar Firebase:
    • Crea un proyecto en la plataforma Firebase.
    • Configura la base de datos en tiempo real para almacenar los datos de temperatura.
    • Obtén las credenciales necesarias (API Key, URL de la base de datos) para conectar tu ESP32 con Firebase.
  6. Verificar el Funcionamiento:
    • Sube el código al ESP32 y verifica que se conecte correctamente a Wi-Fi.
    • Comprueba en la consola de Firebase que los datos de temperatura se estén enviando y almacenando correctamente.
